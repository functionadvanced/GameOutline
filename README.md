# GameOutline
An outline of a text game.

# 可选人物角色：
* 程序员
* 产品经理（待开发）
* 程序员女友（待开发）
* 产品经理女友（待开发）

# 主要情节：
* 程序员求职
* 程序员征友
* 程序员的日常
* 产品经理与程序员

# 游戏过程：
1. NewBee阶段：通过完成大量的练习（puzzle/quiz），来获得技能点以及提升技能点的等级（经验）。
2. Ready For Interview阶段： 如Java/C++/Python达到LV3，或者Java LV2 && C++ LV2 或者前端达到LV3。则可以开始给公司发简历。随机抽取公司开始进行面试。
3. Interview: 相当于Boss版的Ready For Interview题目。
4. 进入公司后：开始有情感交互，程序员与女生，程序员与上司，程序员与产品经理。同时随着时间的推移不断地出现谜题/困难。
5. 提升技能点，赚更多的钱，以及和女生还有manager保持良好的关系。



# 游戏变量：
## 金钱
储蓄：初始化有一定数额的钱（3000），如果是没有工作的状态，金钱按固定比例减少。
工资：找到工作以后，工资与 技能点/技能等级/时间/玩家遇到问题的选择 挂钩。

## 压力值
拼命学习、工作、加班会增加压力值，打游戏、约会等可以减小压力值

## 技能点
  从programming skills中选择。技能有等级（经验），等级（经验）直接与游戏困难程度挂钩，也与获得的金钱挂钩。
  技能从0到lv1需要花费金钱买工具书/网络课程，不同技能花费的金钱不同。

## 时间check point
 总共时间定为100天
 第一章：成长
### 起点 第10天
 刚毕业，暂住在朋友家里。若在10天内未能租到房子，结束。 储蓄:-100 per day
  * 新手教程。
  * 找房子： 找到公寓的概率30% 
  * 学习：压力+20; 技能+;
  * 吃喝玩乐: 压力清零；
  * 泡妞：亲密度+10%；如果连续五天不选择泡妞，亲密度-20%；
 //Bad Ending1：【扫地出门】10天内没租到房子，和朋友吵架，被扫地出门
### 求职 第20天 
 *   储蓄：（房租：-1200/30days） （日常花销：-100 per day） 
 * 学习:技能+;  压力+20
 * 投简历：30%的概率在两天后得到随机一个公司的面试机会  压力+10 --> 玩家选择是否接受面试：一线公司工资1000/7days 40%概率通过 压力+20 / 二线公司工资1500/7days 20%概率通过 压力+30
 * 泡妞：亲密度+20%；压力-20 如果连续五天不选择泡妞，亲密度-20%；
 * 吃喝玩乐：压力清零；
//Bad Ending2：【无业游民】初始化的钱==0，还没找到工作

### 找女友 第30天
 * 带女友吃米其林：亲密度+30% 金钱-500 压力清零
 * 加班：亲密度-10% 压力+20 连续七天不选择加班，游戏结束
 * 下班后打游戏：亲密度-10% 压力清零
 * 告白：假如亲密度>100% 告白成功
//Bad Ending3：【孤独终老】三十天内没有表白成功

### 升职 第40天
如在第40天时没有升职，结束。
 * 加班：亲密度-10% 压力+20 加班两次技能+ 加班三次工资+ 三天不加班同事关系-30%
 * 团队建设： 压力-10 金钱-800 同事关系+30%
 * 吃喝玩乐（和女友）： 压力-20 亲密度+10% 连续三天不选择吃喝玩乐亲密度-20% 
 * 成为Team Leader： 技能>LV3 同事关系>100% 升职成功 工资++ 
//BadEnding2: 【孤独终老】与女友的亲密度==0
//BadEnding3：
第二章：奋斗

### 买车 第50天
如在第50天时没有买车子，结束。
 * 学习理财：储蓄-6000 买车价格-20% 买房价格-10% 得到防骗指南
 * 全组赶Deadline：同事关系-33.3% 压力+40 亲密度-10% 技能+ 一大笔奖金 七天内没有选择这个选项，游戏结束
 * 买车：储蓄-30000 买车以后储蓄三天-500 压力清零 （如没有防骗指南，50%概率储蓄-10000买车失败）
 * 吃喝玩乐（和女友）： 压力-20 亲密度+10% 连续三天不选择吃喝玩乐亲密度-20% 
 
### 买房 第60天
如在第60天时没有交首付，结束。
 * 理财：10%概率储蓄+100% / 5%概率储蓄-95% / 35%概率储蓄+20% / 50%概率储蓄-20% 
 * 全组赶Deadline：同事关系-33.3% 压力+40 亲密度-10% 技能+ 一大笔奖金 七天内没有选择这个选项，游戏结束
 * 买房首付：储蓄-50000 买房以后储蓄五天-6000 压力清零 （如没有防骗指南，50%概率储蓄-30000首付失败）
 * 吃喝玩乐（和女友）： 压力-20 亲密度+10% 连续三天不选择吃喝玩乐亲密度-20% 

### 求婚 第70天
需要买道具戒指。
 * 团队建设： 储蓄-3000 压力-20 同事关系+66.7%
 * 高级吃喝玩乐：储蓄-5000 压力-30 亲密度+30%
 * 全组赶Deadline：同事关系-33.3% 压力+50 亲密度-10% 技能+ 一大笔奖金 七天内没有选择这个选项，游戏结束
 * 理财： 10%概率储蓄+100% / 5%概率储蓄-95% / 35%概率储蓄+20% / 50%概率储蓄-20% 
 
 //事件： 求婚（有房有车且亲密度达到100%）：小戒指5000成功率20%  中戒指10000成功率50% 大戒指20000成功率100%


第三章： 创业


### 开发app 第80天
没有自己独立开发app，不可能有闲暇的资金，结束。

### 投资 第90天
开发的app获得成功，如果没有投资金融产品，结束。
### 投资成功 第100天
投资成功，成为大富翁。



## 情感（Implicit）：

# Bad Ending：
## 扫地出门
10天内没租到房子，和朋友吵架，被扫地出门
## 无业游民
初始化的钱==0，还没找到工作
## 压力过大
压力值>100
## 碌碌无为
没有在规定时间内升职
## 重回原点
升职以后，解决问题的表现太差,或者团队关系==0，被layoff
## 飞来横祸
没在规定时间内买车，等公交车时被车撞飞
## 涉世未深
买房被骗，跳楼
## 孤独终老
三十天内没有表白成功或者表白成功以后亲密度==0
## 渣男一枚
没有在规定时间内求婚
## 财不理你*
没有在规定时间内购买理财产品

# Happy Ending：
## 人生巅峰
买房，买车，创业成功当CEO，迎娶白富美
